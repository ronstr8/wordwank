# Sound Files Directory

Place the following MP3 files in this directory:

- `placement.mp3` - Short, subtle sound for tile placement
- `buzzer.mp3` - Error sound for invalid inputs
- `bigsplat.mp3` - Game end explosion sound
- `ambience.mp3` - Background loop (will play continuously at low volume)

All sounds should be sourced from Freesound.org with proper Creative Commons licensing.
